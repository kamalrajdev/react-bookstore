# React Bookstore

A modern React bookstore app focused on real-world e-commerce workflows.

---

## Features
- Browse and view books
- Add and remove items from cart
- Real-time cart updates
- Responsive UI for all devices
- Search functionality for books

---

## Tech Stack
- React
- JavaScript (ES6+)
- CSS3

---

## Purpose
This project was built to strengthen React development skills by simulating a real-world e-commerce application with reusable components and dynamic UI behavior.

---
